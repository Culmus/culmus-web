Fancy Fonts

If you wish to receive notifications on the new releases in this section, please subscribe to the culmus-announce mailing list.

Note for package maintainers: Each fancy font maintains its own version number. If you wish to package the fonts for a Linux distribution it is recommended that you give your package the version number according to the retrieval date. For example, if the package contans fonts as of September 24, 2023, it could be named culmus-fancy-20230924.rpm.
